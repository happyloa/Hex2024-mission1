## 如何安裝所需套件？

將檔案 clone 下來後，輸入：

    npm install

安裝需要的套件。

接著，輸入：

    npm run dev

在本地端啟動開發環境，預設的監聽端口是 3000，等到本地開發環境正式啟動後，可以在瀏覽器上輸入：

    http://localhost:3000

以在本地端預覽專案
